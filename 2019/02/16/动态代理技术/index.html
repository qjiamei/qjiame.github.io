一.cglib动态代理

  1. 建立接口,定义抽象方法
  2. 建立被代理类实现接口,重写接口里的方法
  3. 建立代理类实现接口MethodInterceptor,重写方法intercept,后面有四个参数
    • Object o
    • Method method
    • Object[] arrays
    • MethodProxy methodProxy
  4. 在代理对象中返回invokeSuper对象 Object invokeSuper = methodProxy.invokeSuper(o, arrays);
  5. 测试类中需要new Enhancer()对象,
  6. 对象中需要设置两个属性enhancer.setSuperclass(Que.class);和enhancer.setCallback(cglib);
  7. 这个enhancer对象使用enhancer.create()方法就可以强转成hose对象,再调用接口里的方法,就实现了

二.jdk动态代理

  1. 写一个接口,接口里写一个抽象方法
  2. 写被代理类,实现接口里的抽象方法
  3. 写jdk代理类,实现InvocationHandler接口,重写invoke方法,里面三个参数
    • Object proxy
    • Method method
    • Object[] paramArrayOfObject
  4. 在jdk代理类传一个object对象;
  5. 使用参数里的mehod.invokef(object,paramArrayOfObject); 得到ivoke对象并返回;
  6. 在测试类中,new一个被代理对象,在new一个代理对象,在他的构造方法里,传参被代理对象,
  7. 使用Proxy.newProxyInstance方法生成对象,此对象可强转成接口对象,
    • que.getClass().getClassLoader()
    • que.getClass().getInterfaces()
    • dkProxy—-(这个是代理对象)
